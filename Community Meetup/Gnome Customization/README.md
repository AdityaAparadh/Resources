- [Daulatrao Patil](https://github.com/RAOdotSH)

## Requirements
- Gnome Desktop Environment
- Gnome Tweak Tool
- Gnome Shell Extensions/Extension Manager

## Installation
```bash
sudo apt update
sudo apt install gnome-tweaks gnome-shell-extensions gnome-shell-extension-manager
```
